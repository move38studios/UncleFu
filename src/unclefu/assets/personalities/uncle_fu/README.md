# Uncle Fu sprites

Drop PNGs here named after each expression in the taxonomy:

```
idle.png  talking.png  disapproving.png  concerned.png
smirk.png  approving.png  alarmed.png
```

Until they exist the menu bar uses emoji fallbacks (😤 😟 😏 😌 😱 🧙‍♂️ 💬).
See `../README.md` for the spec and
[`docs/sprite_design.md`](../../../../../docs/sprite_design.md) for the
Nano Banana Pro prompts and post-process recipe used to bake new sprites.
